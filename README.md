# AI Chatbot User Retention Analysis

> An end-to-end Data Analytics project focused on understanding why users abandon an AI chatbot platform after their initial conversations and providing actionable business insights.

---

## 📌 Project Overview

This project simulates a real-world business scenario where an AI chatbot company wants to understand the factors affecting user engagement, retention, and churn.

The project follows a complete analytics workflow, from raw data collection and cleaning to business analysis and executive dashboard creation.

---

## 🎯 Business Problem

An AI chatbot company has observed that a large percentage of users stop using the product after only a few conversations.

The primary objective of this project is to identify:

- Why users abandon the chatbot
- Which user segments are most likely to churn
- How engagement differs across countries and devices
- Which product features influence retention
- Business recommendations to improve customer retention
