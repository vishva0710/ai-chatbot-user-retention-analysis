# Business Problem

## Project Title

**AI Chatbot User Retention Analysis**

---

# Problem Statement

A growing AI chatbot platform has observed that a significant percentage of users stop using the chatbot after only a few conversations. Although the platform continues to acquire new users, long-term user retention remains low, impacting customer satisfaction, subscription conversions, and overall business growth.

The company wants to understand the reasons behind user abandonment and identify the factors that influence user engagement and retention.

As a Data Analyst, the objective is to analyze user behavior, session activity, conversation history, feedback, and subscription data to uncover actionable insights that can help improve the product experience and increase customer retention.

---

# Business Objective

The primary objective of this project is to identify the key factors that influence user retention and provide data-driven recommendations to improve user engagement and reduce churn.

---

# Business Questions

This project aims to answer the following questions:

1. What percentage of users stop using the chatbot after their first few conversations?

2. Which user segments have the highest churn rate?

3. Does session duration affect user retention?

4. How does conversation quality influence customer satisfaction?

5. Which countries have the highest and lowest retention rates?

6. Which devices are associated with better user engagement?

7. Does subscription status impact retention?

8. Which chatbot features are most frequently used by retained users?

9. Does response time affect user satisfaction?

10. What actions can the company take to improve long-term user retention?


---

# Scope of Analysis

The analysis includes the following datasets:

- Users
- Sessions
- Conversations
- Feedback
- Subscriptions
- Retention Metrics

The project focuses on descriptive and diagnostic analytics to explain user behavior and identify factors contributing to customer churn.
